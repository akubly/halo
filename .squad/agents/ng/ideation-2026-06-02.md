## Ideas

1. **Spatial Audio Beamforming via IMU + Speaker.** Mic input detected from specific head orientation (pitch/roll/heading) combined with accelerometer tap patterns to create a spatial sound "bubble" that follows gaze direction — multi-channel LC3 streams mixed on-device via Lua to make audio feel localized in world-space.

2. **Real-Time Computer Vision Object Tracking Loop.** Camera → libmpix frame buffer → on-device Lua motion detection + centroid tracking → IMU heading sync → display crosshair + compass bearing to tracked object on circular display; closes the sensor loop entirely on-device without host intervention.

3. **Haptic-Like Feedback via Display Flicker Pattern.** Rapid display on/off (240–500 Hz flicker below human perception threshold) creates micro-vibration sensation on glasses frame detected by IMU accelerometer; use for silent alerts, rhythm sync, or tactile feedback without mechanical actuators.

4. **Live Camera Feed Peek Window.** Button press triggers 64×64 camera capture window streamed to display corner (libmpix scaled), allowing real-time peek at what's off-screen before full capture; combines button input + camera + display for instantaneous visual context without full app handshake.

5. **Bluetooth-Tethered Sensor Mesh.** On-device Lua runs Bluetooth Central mode scanner detecting nearby Halo/Frame devices; display shows ID, signal strength, and battery; button cycles through peers for peer-to-peer app discovery and cross-device coordinate passing (heading alignment).

6. **Microphone as Accelerometer Proxy.** Ambient sound volume + frequency envelope from mic analyzed in Lua as a makeshift vibration/impact detector (tap on glasses causes mic spike); use speaker + microphone loop as feedback mechanism for custom percussion-based input protocol.

7. **Dynamic Display Refresh Rate Sync to Head Motion.** IMU gyro senses head rotation speed; Lua adjusts display frame rate (120 Hz smooth tracking vs. 30 Hz static) and camera libmpix output resolution (high detail slow pan, low detail fast turns) to conserve power and match user attention.

8. **Multi-Tap Gesture Recognition with IMU + Button.** Button press + simultaneous IMU pattern (shake, roll, nod) recognized in Lua as compound gestures (2-second window); enables eyes-free command palette: triple-tap+roll = screenshot, long-press+shake = reset, etc.

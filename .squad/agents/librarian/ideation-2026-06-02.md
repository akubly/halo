## Premise
The community is 92% **Gemini Live + camera = answer questions**. The Next Frontier is not "better multimodal transcription" — it's agentic orchestration, persistent identity, and embodied reasoning loops that reshape how glasses mediate human intent.

---

## 8 Untested Patterns for Halo

1. **Persistent Embodied Memory Agents** — Halo maintains a continuous neural profile of you (visual history, voice patterns, emotional tone, decision preferences) updated every session, enabling the glasses to predict intent before you ask.

2. **Peripheral Sense Augmentation (Synesthetic AI)** — The LLM fuses camera + audio + IMU into synthetic sensory channels (e.g., "tactile pressure intensity" via bone conduction, "emotional resonance" as color gradients on HUD), allowing glasses to render abstract concepts as felt/visual metaphors.

3. **Action-Chaining via Ambient Middleware** — Glasses parse scene, reason about desired state, decompose into micro-actions (tap desk, swipe phone, speak command), and execute sequentially without waiting for explicit user prompts between steps.

4. **Local World Model + Planning** — On-device lightweight world model (occupancy grid, object permanence, physics priors) runs inference to predict object motion/human behavior 1–3 seconds ahead, enabling proactive responses before visual input confirms change.

5. **Multiplayer AR Embodied Personas** — Multiple LLM agents inhabit shared AR spaces as glyphs/holograms (dungeon master, therapist, collaborator, adversary) with independent reasoning loops, memory, and personality, competing/cooperating for user attention.

6. **Hallucination-as-Generative Content** — Intentionally surface model uncertainty as artistic/interactive elements (glitchy overlays, morphing UI states, probabilistic object outlines) so user sees the AI's confidence landscape rather than deterministic false answers.

7. **Episodic Context Stitching** — Glasses record micro-moments (visual scenes + ambient audio + emotional tone tags) passively throughout the day, then at night use LLM to stitch fragmented memories into coherent narrative summaries, surfacing patterns you didn't consciously notice.

8. **Agentic Loop with Rollback** — User describes desired outcome (goal); glasses decompose into substeps, predict side-effects via on-device forward model, surface risks to user for approval, execute with undo waypoints at each checkpoint.

---

## Design Principles
- **Lean into wearable constraints**, not away from them (always-on means memory is the currency; microOLED means information density matters)
- **Agentic > Reactive** — User speaks once; glasses anticipate the next 3 moves
- **Embodied > Disembodied** — Render LLM reasoning as spatial/temporal dynamics, not just text
- **Identity > Statelessness** — Glasses know *you*, not just the current query
